## lifo-library

This is a library of components for the [Lifo](https://github.com/lifo123/lifo) project.

## Installation

```bash
npm install @lifo/lifo-library
```

## Usage

```jsx
import React from 'react';
import { Social } from '@lifo/lifo-library';

const App = () => {
  return (
    <div>
      <Social type="github" />
    </div>
  );
